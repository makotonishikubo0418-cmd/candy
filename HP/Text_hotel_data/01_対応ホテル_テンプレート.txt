-------------------------------------------------------------------
description_h1・各subtitle / descriptionのテキスト本文は、改行指定がない場合は、改行無しで設置してください。
-------------------------------------------------------------------
記事内容内の scene（h2）ブロックを上から順に自動採番し、h2に id="scene1" を付与してください。各ブロック直下の subtitle_ / description_ は subtitle_1 / description_1 のように同じ番号を付与してください。以降も同様に scene2, subtitle_2, description_2…で統一してください。scene番号は、記事内のh2（通常scene/鹿児島の人気デリヘル店/周辺スポットなど）を上から順にすべて含めて採番してください。
-------------------------------------------------------------------
template_shop.htmlから、上記店舗の情報を取得して<!-- 店舗名 -->～</li>までを設置。その際、移動時間と交通費は下記記載内容に合わせて修正してください。
-------------------------------------------------------------------
「よくあるご質問」「周辺スポット」は、各セクションのscene番号をメイン番号として使用してください。各項目は subtitle{scene番号}_{連番} / description{scene番号}_{連番} を上から連番で付与してください。例としては、scene5なら「subtitle5_1 / description5_1、subtitle5_2 / description5_2」となります。
-------------------------------------------------------------------
「よくあるご質問」「周辺スポット」の各セクションで、faq-itemの最終ブロックのみ開始タグを <div class="faq-item bd_tb"> にすること。最終ブロック以外は <div class="faq-item bd_t"> のままにすること。
-------------------------------------------------------------------
「よくあるご質問」「基本情報内項目」「料金情報内項目」に情報が書かれてない場合は、<!-- FAQ START -->～<!-- FAQ END --> / 各セクションを削除してください。
-------------------------------------------------------------------
img内、alt="aaaaaaaaaaaaaaaaaaaa"にはホテル名をいれてください。
-------------------------------------------------------------------
各項目の内容を確認の上、必要に応じてHTMLを追加・削除して調整してください。
-------------------------------------------------------------------
最後に、ページ内容に合わせて、<!-- 構造化データ（JSON-LD） START -->も最適化してください。
-------------------------------------------------------------------
基本情報内「URL」は「id="button_1"」「ホテル名（aaaaaaaaaaaaaaaaaaaa）」にリンクを設置すること。
