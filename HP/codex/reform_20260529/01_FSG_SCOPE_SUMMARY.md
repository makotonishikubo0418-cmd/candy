## CANDY SEOサイトへ直接落とさない範囲

以下はFSG改修案内では重要だが、CANDY公開HPを直接編集する前提では扱わない。

| 範囲 | 理由 |
|---|---|
| CTI本体画面 | `candy_HP` ではなく `sauce` / CTI側の実装領域 |
| 売上管理 | 公開HPより業務システム側の機能 |
| 車両CTI / Traccar | CANDY公開HPとは直接別領域 |
| ドライバーマイページ | 公開CANDYサイトの直接改修範囲ではない |
| 女の子マイページ | CANDY公開HPとは別アプリ/別導線の可能性が高い |
| 11店舗共有マイページ | CANDY単体HPだけで決められない |
