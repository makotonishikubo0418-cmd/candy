## デザイン上の問題点

| 問題 | 確認箇所 | 影響 |
|---|---|---|
| placeholder が残っている | `contact.html`、`create.html`、複数エリア、`hotel.html`、テンプレート | 公開時に不自然な表示 |
| 営業時間が不一致 | `system.html` と共通フッター | ユーザー混乱 |
| 公開前 `noindex` | `source\*.html` | 問題点ではなく公開前保護。公開作業時のみ解除確認 |
| `fav.js` が存在しない | 複数ページで参照 | お気に入り機能不具合候補 |
| 画像パス切れ候補 | 複数エリア、`girls.css`、`YTPlayer.css` | 表示崩れ候補 |
| 古い jQuery | CDN jQuery 1.11.2 | セキュリティ・互換性リスク |
| 外部 HTTP リンク | 求人、FC2、グループリンク等 | 混在コンテンツ・信頼性リスク |
| 共通部品が複製 | ヘッダー・フッターが各 HTML に存在 | 修正漏れリスク |
